ℹ️ Algorithm Info
The AllocateRoom algorithm:
1. Filters rooms that are vacant
2. Filters rooms where capacity ≥ students
3. Applies AC filter if required
4. Applies washroom filter if required
5. Picks the room with the smallest capacity (best fit)
6. Marks it as occupied if confirmed
